Sprint One

Repository link

This sprint was really lots of trialling and testing and get feedback from Mike

Reflection

This was the first design and I chose an Italian theme. It was really helpful to assemble the page and get used to working with Boostrap 5. I explored how well the site resized on different screens.
This was all working 'okay' and I spent quite a bit of time exploring features such as media queries and using the root style to set colour variables. I also spent a lot of time trying to get the black line framing the text.
